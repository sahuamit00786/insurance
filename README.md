# InsureMS — Insurance Management System

## Quick Start

### 1. Start the backend
```bash
cd server
npm run dev
```
Server runs on **http://localhost:5000**

### 2. Start the frontend (new terminal)
```bash
cd client
npm run dev
```
App runs on **http://localhost:5173**

### 3. Login
- **Email:** admin@insurance.com
- **Password:** Admin@123

---

## Database
- Host: localhost:3306
- Database: `insurance` (auto-created on first boot)
- User: root / Admin@123
- Migrations run automatically on server start

## Tech Stack
- **Frontend:** React 18 + Vite + Tailwind CSS v4
- **Backend:** Node.js + Express
- **Database:** MySQL 8
